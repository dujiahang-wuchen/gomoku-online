# 五子棋小游戏

一个浏览器版五子棋小游戏，支持本地双人、人机对弈，以及通过房间链接邀请好友对战。

## 功能

- 15x15 五子棋棋盘
- 黑白轮流落子、五连胜负判断、平局判断
- 新局、悔棋、比分和落子记录
- 人机对弈
- 好友房间链接对战
- Cloudflare Tunnel 临时公网链接，方便手机或异地好友访问

## 本地运行

```bash
npm install
npm start
```

打开：

```text
http://localhost:5173
```

## 同一局域网联机

服务器启动后，手机或好友电脑打开你电脑的局域网地址，例如：

```text
http://你的电脑IP:5173
```

进入页面后，房主点击「创建邀请」，把生成的房间链接发给好友，好友打开链接即可加入。

## 公网临时链接

如果手机或好友不在同一网络，可以再开一个终端运行：

```bash
npm run tunnel
```

终端会显示一个 `trycloudflare.com` 链接。双方打开这个公网链接后，房主点击「创建邀请」，把生成的房间链接发给好友。

## 检查代码

```bash
npm run check
```
