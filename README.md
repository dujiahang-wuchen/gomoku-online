# 五子棋小游戏

一个可以在浏览器里玩的小游戏大厅，目前内置五子棋，支持本地双人、人机对弈、好友房间链接对战。

项目第一版已经上线，当前正式部署在 Cloudflare Workers + Durable Objects 上。

正式地址：

```text
https://wuchen-gomoku.cn
```

备用地址：

```text
https://www.wuchen-gomoku.cn
```

打开正式地址后，进入五子棋，点击「创建邀请」，复制生成的房间链接发给好友即可联机。

## 主要功能

- 15x15 五子棋棋盘
- 游客昵称进入和游戏大厅入口
- 大厅支持修改昵称和退出登录，退出不会清空本地棋局或战绩
- 黑棋先手，自动判断五连胜利和平局
- 胜利后高亮连成一线的棋子
- 本地双人对战
- 人机对弈，支持简单、普通、困难三个等级
- 新局、悔棋、比分统计、最近落子记录
- 好友对战中使用申请悔棋机制，对方同意后才会撤销
- 好友对战中再来一局需要对方确认
- 好友房间链接对战
- 好友房间第一局随机分配黑白，之后每局自动交换黑白
- 好友对战胜场按玩家统计，换棋色后胜场仍跟随玩家
- 好友对战优先使用 WebSocket 实时同步
- WebSocket 不可用时自动回退到 HTTP 同步
- 好友房间内支持昵称和文字聊天，刷新后可恢复最近聊天记录
- 好友对战支持手动同步棋局，网络波动后可以一键拉取最新局面
- 刷新页面后会自动恢复本地、人机或好友房间棋局
- 一键复制邀请链接、退出房间、在线人数状态提示
- 好友主动退出房间时会立即提示并更新在线人数
- 好友加入房间时会短暂提示；主动退出、离线或关闭浏览器时，对方会看到常亮提示
- 关闭浏览器后重新打开公网链接会自动回到上次房间
- 对局中会醒目显示“你执黑棋/白棋”
- 支持浅色/黑色主题，黑色主题会同步切换棋盘配色
- 手机布局和触控落点优化
- 手机端支持折叠设置、好友、聊天和落子记录面板
- 支持 Cloudflare Tunnel 生成临时公网链接
- 支持 Cloudflare Workers + Durable Objects 固定公网部署

## 运行环境

需要先安装 Node.js。

建议使用 Node.js 18 或更新版本。

## 本地开发运行

本地运行主要用于开发、调试和临时测试。正式给好友使用时，优先使用上面的正式地址。

第一次运行，先安装依赖：

```bash
npm install
```

启动游戏服务器：

```bash
npm start
```

然后在浏览器打开：

```text
http://localhost:5173
```

## 本机游玩

打开页面后先用游客昵称进入游戏大厅，再选择「五子棋」开始下棋。

- 不开启「人机对弈」：黑白双方在同一台设备上轮流下
- 开启「人机对弈」：玩家和电脑对战
- 可以选择执黑或执白
- 可以选择人机难度：简单、普通、困难

## 悔棋规则

- 本地双人：点击「悔棋」会直接撤销最近一步
- 人机对弈：点击「悔棋」会撤销玩家和电脑各一步
- 好友对战：点击「申请悔棋」会向对方发送申请
- 对方同意后，会撤回到申请方自己的上一手之前，由申请方重新落子
- 对方拒绝后，棋盘保持不变
- 好友对战中每方每局最多可以成功悔棋 3 次
- 好友刷新页面或重新打开邀请链接后，会自动恢复房间里的当前棋局

## 局域网好友对战

如果好友的手机或电脑和你在同一个 Wi-Fi，可以用局域网地址访问。

先查看你电脑的局域网 IP，例如：

```text
192.168.1.23
```

好友打开：

```text
http://你的电脑IP:5173
```

例如：

```text
http://192.168.1.23:5173
```

进入页面后：

1. 房主点击「创建邀请」
2. 页面会生成一个带 `?room=...` 的链接
3. 点击「复制链接」，把这个链接发给好友
4. 好友打开链接后会自动加入房间
5. 第一局会随机分配黑白
6. 之后每次双方同意再来一局，会自动交换黑白

## 公网临时链接

这是备用方案，主要用于本地调试或正式域名临时不可用时应急。正常情况下不需要使用。

如果好友不在同一个 Wi-Fi，可以用 Cloudflare Tunnel 生成临时公网链接。

先保持游戏服务器运行：

```bash
npm start
```

再打开另一个终端运行：

```bash
npm run tunnel
```

终端会显示一个类似这样的链接：

```text
https://xxxx.trycloudflare.com
```

双方都打开这个公网链接。房主点击「创建邀请」，再把生成的房间链接发给好友。

注意：Cloudflare Tunnel 免费临时链接不稳定，电脑关机、网络断开、隧道进程关闭或 Cloudflare 回收后，旧链接就会失效。失效后需要重新运行 `npm run tunnel` 生成新链接。

## Cloudflare Workers 正式部署

当前正式线上版本部署在 Cloudflare Workers。项目内置 `worker.mjs` 和 `wrangler.toml`，好友房间会通过 Durable Objects 保持实时同步。

当前绑定域名：

```text
https://wuchen-gomoku.cn
https://www.wuchen-gomoku.cn
```

`wrangler.toml` 中已经配置了这两个自定义域名，并指向 `gomoku-online` Worker。

首次部署前，先登录 Cloudflare：

```bash
npx wrangler login
```

部署：

```bash
npm run deploy:cloudflare
```

部署成功后，Cloudflare 会更新正式域名上的版本。双方打开正式地址即可创建邀请、复制房间链接并联机对战。

说明：Cloudflare 免费额度适合个人练手和少量好友对战。如果访问量很大，后续需要关注 Cloudflare 账号里的用量限制。

## 常见问题

### 手机打不开局域网地址

常见原因：

- 手机和电脑不在同一个 Wi-Fi
- 电脑防火墙拦截了 `5173` 端口
- 使用了公司网、校园网、公共 Wi-Fi，网络隔离导致设备互相访问不到
- 你发给好友的是 VPN 或虚拟网卡地址，不是真正的 Wi-Fi 地址

可以改用正式地址 `https://wuchen-gomoku.cn`，或者在开发时使用 Cloudflare Tunnel 临时公网链接。

### 公网链接突然不能用了

如果使用的是 `trycloudflare.com` 临时链接，这是正常现象。临时隧道不是固定网址。

重新运行：

```bash
npm run tunnel
```

然后使用新生成的 `trycloudflare.com` 链接。

正式地址 `https://wuchen-gomoku.cn` 不依赖本地电脑和临时隧道，正常情况下不会因为电脑关机而失效。

### 好友对战有延迟

项目会优先使用 WebSocket 实时同步，正常情况下延迟较低。

如果网络环境不支持 WebSocket，会自动回退到 HTTP 同步，延迟会稍高一些。

### 好友加入后看不到落子

可以尝试：

- 双方刷新页面
- 点击「同步棋局」拉取服务器上的最新局面
- 重新创建邀请链接
- 确认两个人打开的是同一个房间链接
- 确认服务器终端没有关闭
- 如果使用临时公网链接，确认该链接还有效

```text
server.js       本地服务器和好友房间同步
tunnel.js       localtunnel 备用脚本
worker.mjs      Cloudflare Workers 好友房间同步
wrangler.toml   Cloudflare Workers 部署配置
package.json    项目命令和依赖
```
